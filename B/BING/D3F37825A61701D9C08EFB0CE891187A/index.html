并行不悖 【bìng xíng bù bèi】

发表于 2011-11-11

基本释义

【释义】 悖：违背；抵触。两种事物同时进行；没有什么冲突。
【正音】 不；不能读作“bú”；悖；不能读作“bó”。
【辨形】 悖；不能写作“勃”。
【语出】 《礼记·中庸》：“万物并育而不相害，道并行而不相悖。”元·脱脱<宋史·赵葵传》：“移忠为孝，臣子之通谊；教孝求忠，君父之至仁。忠孝一原，并行不悖。故日忠臣以事其君，孝子以事其亲，其本一也。”
【英译】 run parallel；carry on two things at the same time without confliction [conflict]；not be mutually exclusive；Both can be implemented without coming into conflict.

用法

紧缩式；作谓语、定语；用于物

例句

我现在所要表达的观点，同我以前所说，本是并行不悖的，怎么能说我自相矛盾呢

近反义词

【近义】 双管齐下、并驾齐驱
【反义】 势不两立、自相矛盾
