本来面目 【běn lái miàn mù】

发表于 2011-11-11

基本释义

【释义】 本来：原来的；面目：面貌。原是佛教用语；指人的本性；后比喻人或事物原来的样子。
【正音】 本；不能读作“bèn”。
【辨形】 本；不能写作“木”。
【语出】 宋·释道原《景德传灯录·袁州蒙山道明禅师》：“祖日：‘不思善，不思恶，正恁么时，阿那个是明上座本来面目！’”
【英译】 true colours； true features

用法

可用于人；也可用于事物。一般作主语、宾语。

例句

他们交往了三十年，直到她死，也没有能看清他的本来面目。

近反义词

【近义】 庐山真面目、真相大白
【反义】 涂脂抹粉、乔装打扮
