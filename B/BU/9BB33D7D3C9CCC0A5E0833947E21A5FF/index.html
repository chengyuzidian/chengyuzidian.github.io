不可名状 【bù kě míng zhuàng】

发表于 2011-11-11

基本释义

【释义】 名：说明。状：描绘，形容。很难用语言形容。
【正音】 状；不能读作“zuànɡ”。
【辨形】 状；不能写作“壮”。
【语出】 晋·葛洪《神仙传·王远》：“衣有文采，又非锦绮，光彩耀目，不可名状。”
【英译】 indescribable；beyond description；beggar description；defy description

用法

常指声音、色彩、感觉等不能用语言来描述。“名”也作“言”。一般作谓语、定语。

例句

站在萧瑟的秋风里，他心中涌起一种不可名状的惆怅。

近反义词

【近义】 不可言宣、不可思议、不可胜言
【反义】 一语破的
