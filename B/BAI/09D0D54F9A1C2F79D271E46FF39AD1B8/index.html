白手兴家 【bái shǒu xīng jiā】

发表于 2011-11-11

基本释义

【释义】 兴:创立。形容原来无基础或条件很差而创出一番事业。同“白手起家”。
【语出】 郭沫若《学生时代·创造十年续篇》：“但是日本人毕竟是白手兴家的苦穷人，他们的钱没有‘洋记’那样的松泛。虽然交际是相当频繁，空气也很乐观，要由日本民间捐助十几二十几万的消息始终在酝酉良着。”
【英译】 start empty-handed； build up from nothing； start from scratch； start from taw

例句

他出国之后白手兴家，创立了一家饭店，之后他就用自己经营饭店赚来的钱资助国内的福利事业。

近反义词

【近义】 白手起家
