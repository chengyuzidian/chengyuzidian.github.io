百不失一 【bǎi bù shī yī】

发表于 2011-11-11

基本释义

【释义】 失：过失，错误。万无一失。形容做事有充分把握，绝不会失误。
【语出】 东汉·王充《论衡·须颂》：“从门应庭，听堂室之言，什而失九；如升堂窥室，百不失一。”
【英译】 There is not a single miss in a hundred tries.

用法

紧缩式；作谓语、定语；指有十足的把握

例句

经过三天三夜的商讨，一个百不失一的方案出笼了。

近反义词

【近义】 万无一失、百不一爽
