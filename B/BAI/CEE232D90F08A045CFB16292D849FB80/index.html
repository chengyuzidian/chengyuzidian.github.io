百般刁难 【bǎi bān diāo nàn】

发表于 2011-11-11

基本释义

【释义】 百般：指采取多种手法；刁难；故意给人为难。用各种手段使对方过不去。
【正音】 难；在这里不能读作“nán”。
【辨形】 刁；不能写作“习”。
【语出】 郭戈《金瓶梅传奇》第十一回：“柔玉不便道出自己身世，编个话儿与她道：‘奴家父母早逝，自幼跟哥嫂度日。只是嫂嫂容不得，百般刁难，与其受人凌辱，倒不如自寻清净，避开人世烦恼。’”
【英译】 create all sorts of obstacles； create obstructions of every description； keep raising objections； put up innumerable obstacles

用法

偏正式；作谓语、定语；形容用各种手段使对方为难

例句

1）在谈判过程中，对方百般刁难，所以未能达成协议。
2）技术员要改革一下生产工序，却遭到了车间主任的百般刁难，说他这是出风头，抬高自己。
3）张驴儿百般刁难窦娥，想让她就范，可窦娥宁死也不愿向这种卑下的小人妥协。

近反义词

【近义】 故意刁难
【反义】 与人方便
