哀哀欲绝 【āi āi yù jué】

发表于 2011-11-11

基本释义

【释义】 哀哀：极度悲哀。 绝：气绝。形容极其悲痛，悲痛得将要气绝。
【正音】 哀；不能读作“ái”。
【辨形】 哀；不能写作“衰”。
【语出】 清·曹雪芹《红楼梦》第13回：“那宝珠按未嫁女之礼在灵前哀哀欲绝。”
【英译】 be so distressed [grieved] as if one’s heart would break

用法

形容极度悲哀；将要气绝。一般作谓语、定语、补语。

例句

1）那安太太进门，一眼便看见姑娘，～的跪在那里……便一直的奔了姑娘去。（清·文康《儿女英雄传》第二十回）
2）她扑在母亲坟前哭得～，众乡邻看见无不为之落泪。

近反义词

【近义】 哀痛欲绝、哀天叫地
【反义】 兴高采烈
