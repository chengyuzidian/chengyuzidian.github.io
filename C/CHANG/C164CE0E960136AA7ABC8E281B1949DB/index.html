长此以往 【cháng cǐ yǐ wǎng】

发表于 2011-11-11

基本释义

【释义】 长期这样下去。多就不好的情况而言。
【正音】 长；不能读作“zhǎnɡ”。
【辨形】 长；不能写作“常”；以；不能写作“已”。
【辨析】 ～和“久而久之”；都有“保持相当长一段时间”的意思。不同在于：①～的“此”是有所指；并多指不好的情况；“久而久之”不一定有所指；并且不论好的、坏的情况都常用。②在表示“长久”的含义上；“久而久之”重于～。
【语出】 鲁迅《二心集·友邦惊诧论》：“友邦人士，莫名惊诧，长此以往，国将不国。”
【英译】 if things go on like this；if things continue this way；persist in so doing

用法

常指时间久了会有不好的结果。一般作分句。

例句

这几年忙于生意，很久不读书了，长此以往，恐怕真的会写不出文章了。

近反义词

【近义】 久而久之、长年累月、天长日久
【反义】 一时半刻
