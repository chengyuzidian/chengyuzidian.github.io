陈陈相因 【chén chén xiāng yīn】

发表于 2011-11-11

基本释义

【释义】 陈：陈旧。因：因袭，沿袭。原指仓库里的存粮一年一年增加，陈粮之上加陈粮。后比喻因袭陈旧事物，役有改进。
【正音】 相；不能读作“xiànɡ”。
【辨形】 陈；不能写作“阵”。
【语出】 汉·司马迁《史记·平准书》：“太仓之粟，陈陈相因，充溢露积于外，至腐败不可食。”
【英译】 follow an old routine；follow a set routine；keep on doing the same thing over and over again；persist in the old ways without any change or improvement

用法

含贬义。比喻沿袭老一套；毫无改进、创新。一般作谓语、宾语、定语。

例句

那陈陈相因的生活，就是过上百年，也毫无意义，毫无价值。

近反义词

【近义】 墨守成规、因循守旧、萧规曹随
【反义】 除旧布新、革故鼎新、推陈出新
