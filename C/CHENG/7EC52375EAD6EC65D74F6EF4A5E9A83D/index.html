称兄道弟 【chēng xiōng dào dì】

发表于 2011-11-11

基本释义

【释义】 指彼此以兄、弟相称，表示情谊深，关系密切。也用于贬义，揩无原则地讲哥们义气。
【正音】 称；不能读作“chèn”。
【辨形】 弟；不能写作“第”。
【语出】 清·李宝嘉《官场现形记》第十二回：“见了同事周老爷一班人，格外显得殷勤，称兄道弟，好不闹热。”
【英译】 address [treat] each other as brothers；address one another in great familiarity；call … brothers；calling each other brothers

用法

多含贬义。形容关系密切。一般作谓语。

例句

别看他们坐在一起称兄道弟，实际上他们根本就是两条路上的人，彼此都心照不宣罢了。

近反义词

【近义】 情同手足、亲如手足
【反义】 行同陌路
